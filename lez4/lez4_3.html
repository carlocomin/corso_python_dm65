🔤 Funzioni – Annotazioni di tipo (Type Hints)

📌 Cosa sono?

Le annotazioni di tipo permettono di specificare il tipo previsto per i parametri e il tipo restituito da una funzione. Non sono obbligatorie e non generano errori a runtime, ma migliorano:

  • ✅ la leggibilità del codice
  • ✅ l'autocompletamento in IDE
  • ✅ la validazione tramite strumenti esterni (es. mypy)

🔹 Sintassi base


def somma(a: int, b: int) -> int:
    return a + b

🔹 Esempi con altri tipi


def saluta(nome: str) -> None:
    print(f"Ciao, {nome}!")

def radice(x: float) -> float:
    return x ** 0.5

🔹 Più tipi complessi


from typing import List, Tuple, Dict

def media(voti: List[int]) -> float:
    return sum(voti) / len(voti)

def coord() -> Tuple[int, int]:
    return (10, 20)

def conta_lettere(frase: str) -> Dict[str, int]:
    ...

❗ Le annotazioni non limitano i tipi accettati: per farlo occorre scrivere controlli espliciti o usare strumenti esterni.

✅ Usa le type hints per rendere il codice più chiaro e affidabile!
