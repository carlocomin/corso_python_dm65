⚡ Funzioni Lambda in Python

📌 Cos'è una funzione lambda?

È una funzione anonima e concisa, definita con la parola chiave lambda.

Si usa per definire piccole funzioni in una singola riga, spesso come argomento di altre funzioni.

🔹 Sintassi


lambda parametri: espressione

🔹 Esempi


# Funzione classica:
def quadrato(x):
    return x * x

# Equivalente lambda:
quadrato = lambda x: x * x
print(quadrato(4))  # ➜ 16
        

somma = lambda a, b: a + b
print(somma(3, 5))  # ➜ 8

🔹 Uso tipico con funzioni come sorted(), map(), filter()


nomi = ["anna", "Luca", "mario"]
nomi_ordinati = sorted(nomi, key=lambda x: x.lower())
print(nomi_ordinati)
        

numeri = [1, 2, 3, 4]
doppi = list(map(lambda x: x * 2, numeri))
print(doppi)  # ➜ [2, 4, 6, 8]
        

pari = list(filter(lambda x: x % 2 == 0, numeri))
print(pari)  # ➜ [2, 4]

✅ Le lambda sono potenti per scrivere codice compatto e leggibile quando la funzione è semplice e usa una sola espressione.

❗ Se la funzione è complessa, preferisci una def normale per chiarezza.
