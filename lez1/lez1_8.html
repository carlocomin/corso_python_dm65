Lezione 1: Conversione e Check dei Tipi in Python

ðŸ”¹ Conversione tra Tipi di Dati

Python permette di convertire i tipi di dati in modo esplicito usando funzioni integrate:


# Conversione di tipi
num_intero = 10
num_stringa = str(num_intero)  # Da int a stringa
num_float = float(num_intero)  # Da int a float

print(num_stringa, type(num_stringa))  # Output: '10' 
print(num_float, type(num_float))  # Output: 10.0

Python converte anche implicitamente alcuni tipi quando necessario, ad esempio nelle operazioni matematiche tra interi e float.

ðŸ”¹ Verifica del Tipo di Dato

Puoi verificare il tipo di una variabile con la funzione type():


variabile = "Hello!"
print(type(variabile))  # Output:

Per controllare se una variabile appartiene a un certo tipo, usa isinstance():


print(isinstance(42, int))  # Output: True
print(isinstance(3.14, float))  # Output: True
print(isinstance("Python", str))  # Output: True
