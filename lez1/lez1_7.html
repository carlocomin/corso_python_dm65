Lezione 1: Operatori Aritmetici Compatti in Python

ðŸ”¹ Cosa sono gli operatori aritmetici compatti?

Gli operatori aritmetici compatti permettono di modificare il valore di una variabile in modo piÃ¹ conciso.

ðŸ”¹ Esempi di utilizzo


# Assegnamento standard
x = 10
x = x + 5  # x ora vale 15

# Utilizzando gli operatori compatti
x = 10
x += 5  # Equivalente a x = x + 5
x -= 3  # Equivalente a x = x - 3
x *= 2  # Equivalente a x = x * 2
x /= 4  # Equivalente a x = x / 4
x //= 2  # Equivalente a x = x // 2
x %= 3  # Equivalente a x = x % 3
x **= 2  # Equivalente a x = x ** 2

print("Valore finale di x:", x)

ðŸ”¹ Spiegazione dettagliata

  • += : Aggiunge un valore alla variabile.
  • -= : Sottrae un valore dalla variabile.
  • *= : Moltiplica la variabile per un valore.
  • /= : Divide la variabile e assegna il risultato (float).
  • //= : Divide la variabile e assegna il risultato arrotondato per difetto.
  • %= : Assegna il resto della divisione.
  • **= : Eleva la variabile alla potenza indicata.

ðŸ”¹ Applicazione pratica


punti = 0
print("Punti iniziali:", punti)

# Incrementiamo i punti con diverse operazioni
punti += 10  # Aggiunge 10 punti
punti *= 2   # Raddoppia i punti
punti -= 5   # Sottrae 5 punti
punti %= 4   # Rende il valore il resto della divisione per 4

print("Punti finali:", punti)
