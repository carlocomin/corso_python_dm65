Lezione 1: Soluzioni degli Esercizi Aggiuntivi

ðŸ”¹ Soluzioni

  1. Divisione intera e resto 
    
a = int(input("Inserisci il primo numero: "))
b = int(input("Inserisci il secondo numero: "))
print("Divisione intera:", a // b)
print("Resto:", a % b)
  2. Estrazione di caratteri da una stringa 
    
parola = input("Inserisci una parola: ")
print("Primo carattere:", parola[0])
print("Ultimo carattere:", parola[-1])
  3. Concatenazione avanzata 
    
parola1 = input("Inserisci la prima parola: ")
parola2 = input("Inserisci la seconda parola: ")
print(parola1 + "-" + parola2)
  4. Stringa con spazi uniformi 
    
parola = input("Inserisci una parola: ")
print(" ".join(parola))
  5. Conversione tra tipi 
    
num = float(input("Inserisci un numero decimale: "))
print("Come intero:", int(num))
print("Come stringa:", str(num))
  6. Stampa con formato fisso 
    
num = int(input("Inserisci un numero: "))
print(f"Numero con 6 cifre: {num:06}")
  7. Formato percentuale 
    
num = float(input("Inserisci un numero tra 0 e 1: "))
print(f"Percentuale: {num:.2%}")
  8. Stampa tabellare di numeri 
    
n1 = int(input("Primo numero: "))
n2 = int(input("Secondo numero: "))
n3 = int(input("Terzo numero: "))
print(f"{n1:8} {n2:8} {n3:8}")
  9. Conversione tra sistemi numerici 
    
num = int(input("Inserisci un numero intero: "))
print("Binario:", bin(num))
print("Ottale:", oct(num))
print("Esadecimale:", hex(num))
  10. Troncamento di una stringa 
    
frase = input("Inserisci una frase: ")
print("Primi 10 caratteri:", frase[:10])
