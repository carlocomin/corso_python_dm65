Lezione 1: Formattazione dei Numeri in Python

ðŸ”¹ Controllo delle Cifre Decimali

Python consente di controllare il numero di cifre decimali con la formattazione delle stringhe.


num = 3.1415926535
print(f"Numero con 2 decimali: {num:.2f}")  # Output: 3.14
print(f"Numero con 4 decimali: {num:.4f}")  # Output: 3.1416

n = 1000000
print(f"Numero formattato: {n:,}")  # Output: 1,000,000 (formato internazionale)
print(f"Numero con punto come separatore: {n:.2f}")  # Output: 1000000.00

ðŸ”¹ Notazione Scientifica

Python permette di rappresentare numeri in notazione scientifica.


n = 0.000123456
print(f"Notazione scientifica: {n:.2e}")  # Output: 1.23e-04
print(f"Notazione scientifica con 4 cifre: {n:.4e}")  # Output: 1.2346e-04

ðŸ”¹ Formattazione con format()


n = 123.456789
print("Numero con 3 decimali: {:.3f}".format(n))  # Output: 123.457
print("Numero con separatore di migliaia: {:,.2f}".format(n))  # Output: 123.46

ðŸ”¹ Allineamento e Padding


n = 42
print(f"Numero allineato a destra: {n:>10}")  # Output: '        42'
print(f"Numero allineato a sinistra: {n:<10}")  # Output: '42        '
print(f"Numero centrato: {n:^10}")  # Output: '    42    '

Questi strumenti permettono di stampare numeri in maniera leggibile e formattata.
