ðŸ”¹ Valutazione della VeritÃ  (Truthy e Falsy Values)

In Python, non solo True e False sono considerati valori booleani, ma anche altri tipi di dati possono essere valutati in un contesto booleano.

Valori considerati False (Falsy)

  • 0 (intero zero)
  • 0.0 (zero float)
  • "" (stringa vuota)
  • [] (lista vuota)
  • () (tupla vuota)
  • {} (dizionario vuoto)
  • None (valore nullo)

Valori considerati True (Truthy)

  • Qualsiasi numero diverso da zero (1, -1, 3.14)
  • Stringhe non vuote ("Python")
  • Liste, tuple e dizionari con almeno un elemento

print(bool(0))      # Output: False
print(bool(123))    # Output: True
print(bool(""))    # Output: False
print(bool("ciao")) # Output: True
