üîπ Gestione degli Eventi

  • Gli eventi rappresentano input o azioni del sistema (es. chiusura finestra).
  • Eventi principali:
    • pygame.QUIT ‚Äì Chiusura finestra.
    • pygame.KEYDOWN ‚Äì Tasto premuto.
    • pygame.KEYUP ‚Äì Tasto rilasciato.
    • pygame.MOUSEBUTTONDOWN ‚Äì Click mouse.
  • Gli eventi sono catturati con pygame.event.get().
for event in pygame.event.get():
    if event.type == pygame.QUIT:
        running = False
    if event.type == pygame.KEYDOWN:
        if event.key == pygame.K_LEFT:
            print("Freccia sinistra premuta")

tasti = pygame.key.get_pressed()
if tasti[pygame.K_RIGHT]:
    x += 5
if tasti[pygame.K_LEFT]:
    x -= 5
