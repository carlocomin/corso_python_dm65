ðŸ”¹ Struttura Base di un Programma Pygame

  • 1. Inizializzazione: pygame.init()
  • 2. Creazione finestra: pygame.display.set_mode()
  • 3. Game loop: gestisce eventi, logica, disegno.
  • 4. Chiusura: pygame.quit()

ðŸ”¹ Esempio Minimo

import pygame

pygame.init()
screen = pygame.display.set_mode((800, 600))
pygame.display.set_caption("Ciao Pygame!")

running = True
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
    screen.fill((173, 216, 230))  # Azzurro pastello
    pygame.display.update()

pygame.quit()
