üîπ Creare una Finestra con Pygame

  • pygame.display.set_mode((larghezza, altezza))
    Crea una finestra della dimensione specificata.
  • pygame.display.set_caption("Titolo")
    Imposta il titolo della finestra.
  • screen.fill((R,G,B))
    Colora lo sfondo con un colore RGB.

üîπ Esempio Completo

import pygame

pygame.init()
screen = pygame.display.set_mode((640, 480))  # Finestra 640x480
pygame.display.set_caption("Finestra di Gioco")

running = True
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    screen.fill((200, 255, 200))  # Verde pastello chiaro
    pygame.display.flip()  # Aggiorna lo schermo

pygame.quit()

üîπ Colori in Pygame

  • I colori sono definiti con tuple (R, G, B).
  • Ogni valore va da 0 a 255.
  • Esempio: (0, 0, 0) = nero, (255, 255, 255) = bianco.
