📚 Lezione 9: Stack (Pila)

🔎 Cos'è uno Stack?

Lo Stack è una struttura dati LIFO (Last In, First Out).
L'ultimo elemento inserito è il primo ad essere rimosso.

⚙️ Operazioni Fondamentali

  • push(x): inserisce un elemento in cima
  • pop(): rimuove e restituisce l'elemento in cima
  • peek(): restituisce l'elemento in cima senza rimuoverlo
  • empty(): verifica se lo stack è vuoto

💻 Implementazione in Python

Usiamo una lista Python per implementare uno Stack:

stack = []
stack.append(10)  # push
stack.append(20)
print(stack.pop())  # 20
print(stack[-1])   # peek: 10
print(len(stack) == 0)  # empty check

💡 Alternativa Migliore:

Usare collections.deque (double-ended queue) è più efficiente:

from collections import deque

stack = deque()
stack.append(10)
stack.pop()
