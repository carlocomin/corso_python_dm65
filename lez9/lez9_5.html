📚 Lezione 9: Queue (Coda)

🔎 Cos'è una Queue?

La Queue (Coda) è una struttura dati FIFO (First In, First Out).
Il primo elemento inserito è il primo ad essere rimosso.

⚙️ Operazioni Fondamentali

  • enqueue(x): inserisce un elemento in coda
  • dequeue(): rimuove e restituisce l'elemento in testa
  • front(): restituisce l'elemento in testa senza rimuoverlo
  • empty(): verifica se la coda è vuota

💻 Implementazione in Python

Usiamo una lista o, meglio ancora, una deque:

from collections import deque

queue = deque()
queue.append(10)   # enqueue
queue.append(20)
print(queue.popleft())  # dequeue
print(queue[0])        # front
print(len(queue) == 0) # empty check

💡 Varianti delle Code

  • Double Ended Queue (Deque): inserimenti/rimozioni da entrambe le estremità
  • Coda Circolare: coda con gestione di spazio fisso (in ambienti embedded)
