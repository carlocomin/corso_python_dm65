🌳 Lezione 9: Alberi Binari

📌 Cos'è un Albero Binario?

Un Albero Binario è una struttura dati gerarchica in cui ogni nodo ha al massimo due figli: sinistro e destro.

👨‍💻 Definizione in Python

class Nodo:
    def __init__(self, valore):
        self.valore = valore
        self.sinistro = None
        self.destro = None

🌿 Tipi di Visite Ricorsive

  • Preorder: radice → sinistro → destro
  • Inorder: sinistro → radice → destro
  • Postorder: sinistro → destro → radice
def inorder(nodo):
    if nodo:
        inorder(nodo.sinistro)
        print(nodo.valore)
        inorder(nodo.destro)

🛠️ Costruzione Manuale

root = Nodo(10)
root.sinistro = Nodo(5)
root.destro = Nodo(15)
