🧮 Lezione 9: HashMap (dict) e Set

📌 HashMap (dict)

Una HashMap (in Python: dict) associa chiavi a valori.

mappa = {"nome": "Mario", "età": 30}
print(mappa["nome"])  # Mario

# Inserimento / modifica
mappa["città"] = "Roma"
mappa["età"] += 1

# Iterazione
for chiave in mappa:
    print(chiave, mappa[chiave])

⚙️ Operazioni utili

  • in: verifica esistenza chiave
  • get(chiave, valore_default)
  • items(), keys(), values()

🧩 Set: insiemi non ordinati e unici

s = {1, 2, 3}
s.add(4)
s.remove(2)
print(3 in s)  # True

🔁 Operazioni tra insiemi

  • a | b: unione
  • a & b: intersezione
  • a - b: differenza
