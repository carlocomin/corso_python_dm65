⚡ Divide et Impera – Esempio: Quick Sort

🎯 Problema

Ordinare una lista di numeri interi utilizzando Divide et Impera, con partizionamento attorno a un pivot.

🔧 Codice Python

def quick_sort(lista):
    if len(lista) <= 1:
        return lista

    pivot = lista[0]
    minori = [x for x in lista[1:] if x < pivot]
    maggiori = [x for x in lista[1:] if x >= pivot]

    return quick_sort(minori) + [pivot] + quick_sort(maggiori)

📝 Analisi Divide et Impera

  • Divide: Scelgo un pivot e partiziono in minori e maggiori
  • Conquer: Ordino ricorsivamente i due sottoproblemi
  • Combine: Concateno risultato finale

📈 Complessità Temporale

  • Migliore Caso (pivot centrale): O(N log N)
  • Medio Caso (pivot casuale): O(N log N)
  • Peggiore Caso (pivot minimo o massimo): O(N²)
