🔎 Divide et Impera – Esempio: Ricerca Binaria

🎯 Problema

Data una lista ordinata, determinare se un elemento x è presente, usando Ricerca Binaria.

🔧 Codice Python

def ricerca_binaria(lista, x, sin, des):
    if sin > des:
        return -1  # Elemento non trovato

    centro = (sin + des) // 2

    if lista[centro] == x:
        return centro
    elif x < lista[centro]:
        return ricerca_binaria(lista, x, sin, centro-1)
    else:
        return ricerca_binaria(lista, x, centro+1, des)

📝 Analisi Divide et Impera

  • Divide: Divido la lista in metà ad ogni passo
  • Conquer: Chiamata ricorsiva sulla metà rilevante
  • Combine: Nulla da combinare (è un problema di decisione)

📈 Complessità Temporale

Ad ogni passo il problema si riduce di metà → Complessità: O(log N)
