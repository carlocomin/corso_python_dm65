üî¢ Fibonacci Ricorsivo

def fibonacci(n):
    if n == 0:
        return 0
    if n == 1:
        return 1
    return fibonacci(n-1) + fibonacci(n-2)

üîã Potenza Ricorsiva

def potenza(base, esponente):
    if esponente == 0:
        return 1
    return base * potenza(base, esponente-1)

üîç Ricerca Binaria Ricorsiva

def ricerca_binaria(lista, x, sinistra, destra):
    if sinistra > destra:
        return -1  # Elemento non trovato
    centro = (sinistra + destra) // 2
    if lista[centro] == x:
        return centro
    if x < lista[centro]:
        return ricerca_binaria(lista, x, sinistra, centro-1)
    return ricerca_binaria(lista, x, centro+1, destra)

‚úÖ Considerazioni

  • Fibonacci mostra un limite della ricorsione "naive"
  • Potenza e Ricerca Binaria sono esempi eccellenti di ricorsione efficiente
  • Ricerca Binaria funziona solo su liste ordinate!
