üîπ Cosa sono?

Gli operatori speciali sono metodi predefiniti con doppio underscore (es. __str__, __add__) che permettono di definire il comportamento di oggetti personalizzati in operazioni comuni (stampa, confronto, somma...).

üî∏ Alcuni metodi magici comuni

  • __init__(self): costruttore
  • __str__(self): stringa leggibile per print()
  • __repr__(self): rappresentazione tecnica
  • __add__(self, other): somma con +
  • __eq__(self, other): confronto con ==
  • __len__(self): usato da len(obj)

üîß Esempio pratico

class Punto:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __str__(self):
        return f"Punto({self.x}, {self.y})"

    def __add__(self, other):
        return Punto(self.x + other.x, self.y + other.y)

    def __eq__(self, other):
        return self.x == other.x and self.y == other.y

p1 = Punto(1, 2)
p2 = Punto(3, 4)
p3 = p1 + p2  # chiama __add__
print(p3)     # ‚ûú Punto(4, 6)
print(p1 == p2)  # ‚ûú False
