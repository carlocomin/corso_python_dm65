🧬 Overriding dei Metodi in Python

🔹 Cos'è l'overriding?

L’overriding consiste nel ridefinire un metodo ereditato da una superclasse in una sottoclasse. Permette di modificare o estendere il comportamento ereditato.

🔸 Esempio base

class Animale:
    def parla(self):
        print("Verso generico")

class Cane(Animale):
    def parla(self):
        print("Bau!")

a = Animale()
c = Cane()
a.parla()  # ➜ Verso generico
c.parla()  # ➜ Bau!

🔸 Uso di super() per estendere il metodo base

class LoggingCane(Cane):
    def parla(self):
        print("[LOG] Il cane sta per abbaiare")
        super().parla()  # chiama parla() di Cane

✔ Con super() è possibile richiamare il metodo originale della classe base e poi aggiungere comportamento specifico.

⚠️ Note importanti

  • I nomi dei metodi devono coincidere per sovrascrivere
  • Python applica l’MRO (Method Resolution Order) in caso di ereditarietà multipla
  • Non c’è bisogno di specificare “override” come in C++: è implicito
