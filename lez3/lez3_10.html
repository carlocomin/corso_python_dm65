Lezione 3: 🔍 Liste in Python – Slice, Copie, Liste Annidate e Tipi Misti

🔹 Slice e Copie

Lo slicing permette di accedere a una porzione della lista:

lista = ["uno", "due", "tre", "quattro", "cinque"]
sublista = lista[1:4]   # ➜ ['due', 'tre', 'quattro']

È possibile creare una copia completa della lista:

copia = lista[:]  # Copia indipendente dell'intera lista

Si possono anche sostituire più elementi via slice:

lista[:3] = ["a", "b", "c"]
# ➜ ['a', 'b', 'c', 'quattro', 'cinque']

🔹 Liste Annidate

Le liste possono contenere altre liste (come matrici):

matrice = [
    [1, 2],
    [3, 4],
    [5, 6]
]
print(matrice[1][0])  # ➜ 3

Si può accedere a righe e colonne come in una tabella.

🔹 Tipi Misti e Liste Vuote

Le liste in Python possono contenere elementi di tipi diversi:

mista = [1, "ciao", True, 3.14]
for x in mista:
    print(type(x), x)

Per creare una lista vuota da riempire successivamente:

v = []
v.append("nuovo elemento")

Una lista vuota ha lunghezza 0 e può essere usata come accumulatore.
