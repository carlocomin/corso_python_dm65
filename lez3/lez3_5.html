Lezione 3: 🔁 Iterare su una Lista con il ciclo for

📌 Sintassi

for elemento in lista:
    # blocco di codice
    print(elemento)

Con il ciclo for, puoi scorrere una lista elemento per elemento in modo semplice ed efficace. Python gestisce internamente l'indice, rendendo il codice più leggibile e meno soggetto a errori.

✅ Esempio base

frutti = ["mela", "banana", "ciliegia"]
for frutto in frutti:
    print("Mi piace la", frutto)

📌 Iterare su liste miste

dati = [1, "ciao", 3.14, True]
for valore in dati:
    print(type(valore), "->", valore)

🎯 Uso combinato con enumerate()

Se vuoi anche l'indice dell'elemento:

nomi = ["Alice", "Bob", "Clara"]
for i, nome in enumerate(nomi):
    print(f"{i}: {nome}")

🚫 Attenzione: modificare una lista durante l'iterazione

Modificare la lista che stai iterando può produrre risultati imprevisti. Se serve, crea prima una copia:

for x in lista[:]:  # copia della lista
    if x < 0:
        lista.remove(x)

Iterare su una lista è uno degli strumenti più potenti e usati in Python!
