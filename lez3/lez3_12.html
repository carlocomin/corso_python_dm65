Lezione 3: 📦 Tuple in Python – Approfondimento

🔹 Immutabilità: una protezione utile

t = (1, 2, 3)
# t[0] = 100  ❌ Errore: TypeError!

✅ Perfette per dati che non devono cambiare.

🔹 Tuple annidate

studenti = (
    ("Alice", 25),
    ("Bob", 22),
    ("Carla", 27)
)
for nome, età in studenti:
    print(f"{nome} ha {età} anni")

🔹 Uso come chiavi di un dizionario

coordinate = {}
coordinate[(2, 3)] = "albero"
coordinate[(0, 0)] = "partenza"
print(coordinate[(2, 3)])  # ➜ albero

✅ Solo oggetti immutabili possono essere usati come chiavi.

🔹 Conversione tra lista e tupla

lista = [1, 2, 3]
tupla = tuple(lista)
lista2 = list(tupla)

🔹 Tuple e funzioni

def coordinate():
    return (45.0, 12.3)

lat, lon = coordinate()
print(f"Latitudine: {lat}, Longitudine: {lon}")

✅ Le tuple sono ideali per restituire più valori.

📌 Ricorda

  • Le tuple sono ordinate e immutabili
  • Supportano slicing, unpacking, `in`, `len()`, `count()`, `index()`
  • Utili in strutture dati e performance-critical code
