Lezione 3: 🔁 Iterazione Avanzata su Liste

📌 Iterare su una lista ed eseguire trasformazioni

nomi = ["anna", "luca", "giulia"]
for nome in nomi:
    print(nome.capitalize())  # ➜ Anna, Luca, Giulia

🔢 Iterare su una lista di numeri e applicare una formula

numeri = [1, 2, 3, 4, 5]
for n in numeri:
    quadrato = n ** 2
    print(f"{n}² = {quadrato}")

📍 Iterare su una lista con condizione (filtraggio)

parole = ["ciao", "mela", "cane", "gatto", "cielo"]
for parola in parole:
    if parola.startswith("c"):
        print(parola)  # Solo parole che iniziano per "c"

🧮 Iterare su una lista con somma cumulativa

valori = [100, 200, 50]
totale = 0
for v in valori:
    totale += v
print("Totale:", totale)

📚 Iterare su una lista di tuple

studenti = [("Alice", 9), ("Bob", 7), ("Clara", 8)]
for nome, voto in studenti:
    print(f"{nome} ha preso {voto}/10")

📋 Iterare su una lista al contrario

messaggio = ["Fine", "quasi", "è", "lezione", "La"]
for parola in reversed(messaggio):
    print(parola, end=" ")  # ➜ La lezione è quasi Fine

⚠️ Modificare la lista durante l’iterazione (con copia)

numeri = [1, -2, 3, -4, 5]
for n in numeri[:]:  # copia della lista
    if n < 0:
        numeri.remove(n)
print(numeri)  # ➜ [1, 3, 5]

🧠 Iterare su liste permette di costruire logiche complesse: filtrare, trasformare, contare, accumulare, confrontare… tutto parte da qui.
